Text style transfer Evaluation Test

実験に協力して頂き、ありがとうございます。
本実験ではスタイル変換した日本語テキストについて、3項目を4点もしくは5点満点で評価して頂きます。
元テキストとモデルから生成された変換テキストの両方を読み、以下の評価を行ってください。
今回取り扱うスタイルは「標準語・関西弁」です。
実験の手順を説明します。よく読んでからお進みください。

実験手順

  1. 名前を入力し、関西弁がどれくらい分かるか、以下の4段階で一番近いものを選んでください。
      2. 1. 関西弁にほとんど触れたことが無いレベル
      2. メディア上での知識などで少し分かるレベル
      3. 関西圏での生活経験や，親族に関西弁ネイティブがいるなどの理由である程度分かるレベル
      4. 関西圏出身のネイティブ、もしくは関西圏での長い生活経験でネイティブと同等のレベル
  2. 1から5までのセット番号を選んで下さい。
  3. "Start Experiment"を押して下さい。
  4. 元テキストと生成テキストを読み、相応しいと思った評価を選択して下さい。
    本評価実験では、生成されたテキストについて、
    「スタイル変換度合い」標準語・関西弁が変換されているか
    「コンテンツ保存性」：元テキストの内容が保存されているか
    「自然性」：日本語として自然か否か
    の3つの観点から評価をしてもらいます。
  5. 選択を終えたら"next"ボタンを押し、次の実験に進んで下さい。
  6. 評価実験を全て終えたら"finish"ボタンを押して結果をダウンロードして下さい。
    結果は吉岡のSlackか下記のメールアドレスに送信して下さい。
  7. Emailアドレス: yoshioka.daiki@g.sp.m.is.nagoya-u.ac.jp

評価時の注意

  • スタイルの変換方向は2種類（関西弁→標準語と、標準語→関西弁）で、順番はランダムです。
  • 実験の途中でも、インターネット検索などを利用して頂いて構いません。

その他注意

  • この実験は10～20分ほど要します。
  • 実験中はページの再読み込みは行わないで下さい（始めからになってしまいます）。
  • スマートフォンでは結果をダウンロードできない場合があるのでPCを使用して下さい。
  • お時間のある方は複数セットで実験して頂けると大変助かります。
  • 別のセットで評価を始める場合は、お手数ですがページの更新を行うかURLを開きなおしてください。
  • 評価ページの不備や質問等ありましたら、
    吉岡のSlackかメールアドレス[yoshioka.daiki@g.sp.m.is.nagoya-u.ac.jp]にご連絡ください。
